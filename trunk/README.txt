DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/    (works with 2.4 or 2.5)
  PyGame:     http://www.pygame.org/

    Note: some GNU/Linux users have reported visualization errores while 
    running the game with python 2.5. If you do so, run it with python 2.4.

RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.py" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py


HOW TO PLAY THE GAME:

    Move round the screen with the arrow keys.
    A key and Space Bar twist you in a way and 
    S key in the other way.

Bottles:
    - green: LOST energy
    - blue: WIN energy
    - yellow: WIN A LOT of energy
    - red: destroy ALL spells in the screen
